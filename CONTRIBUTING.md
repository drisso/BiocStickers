# Contributing to BiocStickers

Thank you for your interest in contributing to the BiocStickers repository! We
welcome all contributions of stickers related to Bioconductor packages and
projects.

## How to Contribute

1. Fork this repository.
2. Create a new branch for your sticker.
3. If contributing a conference sticker, create a new folder in the `events/`
   directory with the name of the conference (if one does not already exist).
4. Most stickers are in PNG format with transparent backgrounds but PDF and SVG
   formats are also accepted.
5. Add a `README.md` file to your sticker's folder with the following:
    - Sticker name
    - Designer's name and sticker maintainer's name (if different)
    - License information
    - Preview image of the sticker (with ~200px height)
    - Optionally, a brief description of the sticker
6. Update the `README.md` file in the main repository folder to include your
   new sticker.
7. Submit a pull request.

## Sticker Guidelines

- Stickers should be directly related to Bioconductor packages or projects.
- Stickers are predominantly in PNG format but stickers in PDF and SVG
  formats are also accepted.
- Stickers should have a transparent background.
- Sticker PNG height should be 5 cm and resolution should be at least 300 dpi.
- Stickers should be licensed with a CC-BY-type license. Otherwise, CC0 1.0
  Universal Public Domain License applies by default.
